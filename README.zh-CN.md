## 🧩 功能矩阵

| 模块 | 能力 |
| --- | --- |
| 魔方阶数 | 3×3、4×4、5×5 |
| 旋转类型 | 外层、内层、宽层、三层宽转、中层、整体旋转 |
| 交互方式 | 鼠标拖拽、触摸拖拽、贴纸射线拾取、键盘快捷键 |

| 计时系统 | 按住准备、全屏计时、PB、Ao5、Ao12、TPS |
| 智能还原 | Kociemba 风格 3×3 还原序列与分步引导 |
| 视觉表现 | WebGL 场景、玻璃 UI、霓虹光环、粒子爆发、宇宙网格 |
| 本地存储 | 使用 `localStorage` 保存成绩与个人统计 |
| 构建方式 | 零构建步骤，纯 HTML / CSS / JavaScript |

## 🛠️ 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/muskke/apexcube-cosmic-lab.git
cd apexcube-cosmic-lab
```

### 2. 直接打开

用浏览器打开 `index.html`。

```txt
index.html
```

### 3. 或使用本地静态服务

推荐在测试 CDN、浏览器策略或 GitHub Pages 部署行为时使用。

```bash
python3 -m http.server 5173
```

然后访问：

```txt
http://localhost:5173
```

## 🎮 操作方式

### 鼠标与触摸

| 操作 | 效果 |
| --- | --- |
| 拖动空白区域 | 旋转观察视角 |
| 拖动魔方贴纸 | 旋转对应魔方层 |
| 鼠标滚轮 | 放大或缩小视角 |
| 触屏拖动 | 移动端友好的魔方交互 |

### 键盘快捷键

| 输入 | 效果 |
| --- | --- |
| `U D L R F B` | 外层旋转 |
| `Shift + U/D/L/R/F/B` | 外层逆向旋转 |
| `Alt + U/D/L/R/F/B` | 内层旋转 |
| `Ctrl + U/D/L/R/F/B` | 双层宽转 |
| `Ctrl + Alt + U/D/L/R/F/B` | 5×5 三层宽转 |
| `M E S` | 中层旋转 |
| `X Y Z` | 整体旋转 |
| `Space` | 计时器准备 / 开始 / 停止 |
| `Esc` | 取消计时 |

## 🏗️ 技术架构

```txt
ApexCube Cosmic Lab
├── Scene System 场景系统
│   ├── Camera 相机
│   ├── Lighting 灯光
│   ├── Renderer 渲染器
│   ├── Cosmic dust 宇宙星尘
│   └── Neon aura 霓虹光环
├── Cube Engine 魔方引擎
│   ├── Cubie generation 小方块生成
│   ├── Sticker generation 贴纸生成
│   ├── Physical grid state 物理网格状态
│   └── State snapping 状态归整
├── Move System 旋转系统
│   ├── Standard turns 标准旋转
│   ├── Inner-layer turns 内层旋转
│   ├── Wide-layer turns 宽层旋转
│   ├── Middle-layer turns 中层旋转
│   ├── Whole-cube rotations 整体旋转
│   └── Notation parser 公式解析器
├── Interaction Layer 交互层
│   ├── Raycasting 射线拾取
│   ├── Sticker picking 贴纸识别
│   ├── Local drag resolution 局部拖拽解析
│   └── Layer mapping 层映射
├── Speed Layer 速拧层
│   ├── Non-blocking queue 非阻塞队列
│   ├── Adaptive acceleration 自适应提速
│   ├── Move history 步骤历史
│   └── Undo workflow 撤回流程
├── Practice Layer 练习层
│   ├── WCA-style timer WCA 风格计时器
│   ├── PB / Ao5 / Ao12
│   ├── TPS statistics TPS 统计
│   └── Recent solve records 最近成绩
└── Smart Solve Layer 智能还原层
    ├── Kociemba pipeline 求解流程
    ├── Solve sequence 还原序列
    ├── Guided move steps 分步引导
    └── Progress feedback 进度反馈
```
